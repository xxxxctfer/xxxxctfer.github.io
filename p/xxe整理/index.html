еҚұе®і

1гҖҒж–Үд»¶иҜ»еҸ–пјҡе®ҡд№үдёҖдёӘеҢ…еҗ«ж–Үд»¶еҶ…е®№зҡ„еӨ–йғЁе®һдҪ“пјҢз„¶еҗҺеңЁеә”з”Ёзҡ„е“Қеә”йҮҢиҝ”еӣһе®ғгҖӮ 2гҖҒSSRFж”»еҮ»пјҡжҠҠеӨ–йғЁе®һдҪ“е®ҡд№үжҲҗдёҖдёӘжҢҮеҗ‘еҗҺз«Ҝзі»з»ҹзҡ„URLпјҢи®©жңҚеҠЎеҷЁеҺ»и®ҝй—®е®ғ 3гҖҒеёҰеӨ–ж•°жҚ®жі„йңІпјҡжҠҠжңҚеҠЎеҷЁдёҠзҡ„ж•Ҹж„ҹж•°жҚ®еҒ·еҒ·ең°дј еҲ°й»‘е®ўжҺ§еҲ¶зҡ„зі»з»ҹдёҠгҖӮ 4гҖҒй”ҷиҜҜж¶ҲжҒҜжі„йңІпјҡи§ҰеҸ‘дёҖдёӘеҢ…еҗ«ж•Ҹж„ҹж•°жҚ®зҡ„XMLи§Јжһҗй”ҷиҜҜж¶ҲжҒҜгҖӮ

дҝ®еӨҚ

  1. жңҖзӣҙжҺҘзҡ„дҝ®еӨҚпјҡзҰҒз”ЁеӨ–йғЁе®һдҪ“еҠ иҪҪеҷЁгҖӮеңЁи§Јжһҗ XML д№ӢеүҚи°ғз”Ё libxml_disable_entity_loader(true)гҖӮиҝҷдёӘеҮҪж•°дјҡе…ЁеұҖзҰҒжӯўжүҖжңү libxml ж“ҚдҪңеҠ иҪҪеӨ–йғЁе®һдҪ“гҖӮ
  2. й’ҲеҜ№ PHP 8.0+ зҡ„жҺЁиҚҗдҝ®еӨҚпјҡдҪҝз”Ё libxml_set_external_entity_loader()
  3. дҪҝз”Ёе®үе…Ёзҡ„и§ЈжһҗйҖүйЎ№пјҡеңЁ loadXML() жҲ– load() ж—¶жҢҮе®ҡеҗҲйҖӮзҡ„йҖүйЎ№пјҢеҸҜд»ҘйҒҝе…ҚеҠ иҪҪеӨ–йғЁе®һдҪ“пјҢдҪҶйңҖиҰҒз»“еҗҲ libxml_disable_entity_loader жҲ–е®һдҪ“еҠ иҪҪеҷЁжүҚиғҪеҪ»еә•йҳ»ж–ӯ
  4. е®Ңе…ЁзҰҒз”Ё DTD и§Јжһҗ
  5. дёҖдәӣе…¶д»–ж–№ејҸпјҡ иҫ“е…ҘйӘҢиҜҒпјҡеҰӮжһңеҸҜиғҪпјҢеҜ№ XML з»“жһ„иҝӣиЎҢзҷҪеҗҚеҚ•йӘҢиҜҒпјҢжӢ’з»қеҢ…еҗ« жҲ–еӨ–йғЁе®һдҪ“зҡ„иҜ·жұӮгҖӮ жңҖе°ҸжқғйҷҗеҺҹеҲҷпјҡзЎ®дҝқ Web жңҚеҠЎеҷЁиҝҗиЎҢиҙҰжҲ·еҜ№ж–Үд»¶зі»з»ҹеҸӘжңүжңҖе°Ҹеҝ…иҰҒжқғйҷҗпјҢйҳІжӯўиҜ»еҸ–ж•Ҹж„ҹж–Үд»¶гҖӮ дҪҝз”Ёжӣҝд»Јж јејҸпјҡеҰӮжһң XML дёҚжҳҜеҝ…йЎ»зҡ„пјҢеҸҜиҖғиҷ‘дҪҝз”Ё JSON зӯүдёҚжҳ“еҸ—еҲ°е®һдҪ“ж”»еҮ»зҡ„ж•°жҚ®ж јејҸгҖӮ
